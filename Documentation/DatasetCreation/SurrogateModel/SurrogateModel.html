B2BDC.B2Bmodels Package

Introduction

The B2BDC.B2Bmodels package includes all classes and methods relative to surrogate models. A superclass B2BDC.B2Bmodels.Model is used to contain all common properties and methods a surrogate model has. A surrogate model is a functional representation of the quantity of interest (QOI). The model can be a quadratic function, a rational quadratic function or a high order polynomial function.

Tutorial

There are two major ways to generate a B2BDC.B2Bmodels.Model object. User can generate a specific surrogate model by providing the necessary variable and coefficient information. A tutorial of how to generate surrogate model by variable and coefficient information can be found here. User can also generate a specific surrogate model by providing the necessary variable information and the corresponding simulation data. The B2BDC toolbox has its fitting functions that can fit the provided data with the specified surrogate model. A tutorial of how to generate surrogate model by data can also be found.
